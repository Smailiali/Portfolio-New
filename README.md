## `// ABOUT`

Personal developer portfolio built from scratch — no templates, no page builders. Designed with a terminal-meets-editorial aesthetic to showcase projects and experience through interactive elements rather than static content. Every design decision serves one goal: land a software engineering role.

---

## `// FEATURES`

- 🖥️ &nbsp;Terminal-themed boot sequence on every page load
- ⌨️ &nbsp;VS Code-style command palette (`Ctrl+K` / `⌘K`)
- 🎮 &nbsp;Konami Code easter egg with Matrix rain animation
- ✨ &nbsp;Typing animation hero with live code editor mockup
- 📊 &nbsp;Interactive skill constellation with hover interactions
- 🗂️ &nbsp;Horizontal interactive timeline for work experience
- 🃏 &nbsp;3D-tilting project cards with custom SVG mockups
- 📱 &nbsp;Fully responsive with mobile-specific fallbacks
- ♿ &nbsp;Accessible — `prefers-reduced-motion`, ARIA labels, keyboard navigation
